# Examples

These examples are intentionally small. Use them as patterns, not as full application templates.

## minimal-app

Shows `AddHeimdall`, `UseHeimdall`, `MapHeimdallPage`, a layout, and a simple page.

## form-content-action

Shows a form wired with `.Heimdall().Submit(...)`, payload binding with `[ContentPayload]`, and a content action that returns a replacement HTML fragment.

## live-sse-feed

Shows a Bifrost SSE subscription and server-side publication of HTML fragments.
